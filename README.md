## Particle-Frontend-React
particle (personal blog site) frontend with react.js library

# versions
- react.js: 18.2.0
- node.js: v20.14.0
- npm: 10.7.0

# Setup instruction 
Clone the repo:
```
git clone https://github.com/dheeraaz/Particle-Frontend-React.git
cd Particle-Frontend-React
npm install
npm run dev
```
